# Portfolio – React

Portfolio personnel réalisé avec **React** afin de présenter mes projets et de mettre en pratique mes compétences en développement front-end.

Ce projet m’a permis de travailler sur la navigation, les animations, la gestion du scroll et l’intégration d’interfaces modernes et responsives.

Projet, en cours de construction

---

## ✨ Fonctionnalités

- Page d’accueil
- Page projets
- Page contact
- Navigation fluide avec React Router
- Animations et effets de scroll
- Sliders interactifs
- Design responsive

---

## 🛠️ Technologies utilisées

- **React**
- **React Router DOM**
- **Sass (SCSS)**
- **Framer Motion**
- **Swiper.js**
